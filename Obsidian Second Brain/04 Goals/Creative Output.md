# Creative Output

Create proof that your learning is real.

Output ideas:

- Short posts about AI workflows
- Tiny coding projects
- Before/after workflow demos
- Notes turned into guides
- Visual explainers
- Personal experiments

Creation loop:

- Capture in [[Inbox]]
- Develop in [[Content and Creation Project]]
- Review with AI
- Publish or archive

## Output Ladder

Low friction:

- Private note
- Checklist
- Prompt
- Tiny demo

Medium:

- Tutorial
- Short video script
- Blog post
- GitHub project

High:

- Productized workflow
- Course/module
- App
- Public challenge

## Taste Builder

For anything you create, ask:

- What is the promise?
- Who is it for?
- What is the useful takeaway?
- What should be removed?
- What would make this feel more like me?
