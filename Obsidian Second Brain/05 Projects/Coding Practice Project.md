# Coding Practice Project

Outcome: improve coding skill with AI as a tutor and reviewer.

Practice ideas:

- Build a small web app
- Write Python utilities
- Refactor old code
- Add tests to an existing project
- Ask AI to explain unfamiliar code

Workflow:

- Use [[AI Coding Workflow]]
- Track skills in [[Skill Tree]]
- Save reusable prompts in [[AI Workflow Library]]

## Practice Ladder

Beginner reps:

- Read a file and explain it
- Change one UI element
- Fix one bug
- Add one small function
- Write one test

Intermediate reps:

- Build a small app end-to-end
- Add persistence
- Add authentication
- Add API calls
- Refactor a messy module

AI-assisted drills:

- Ask AI to explain code, then verify by editing it.
- Ask AI for a test plan, then implement one test yourself.
- Ask AI for three approaches, then choose and justify one.
