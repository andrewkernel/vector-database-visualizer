## Practical Takeaways

- Model lineups change, so model-specific advice should be refreshed.
- Choose models based on job: reasoning, speed, cost, multimodal ability, tool use, and output needs.
- Use specialized capabilities such as web search, file search, image, audio, and structured output when the task requires them.
