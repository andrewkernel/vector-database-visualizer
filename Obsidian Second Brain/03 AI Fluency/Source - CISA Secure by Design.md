# Source - CISA Secure by Design

Source: CISA, Secure by Design.  
URL: https://www.cisa.gov/securebydesign  
Accessed: 2026-06-24

## Practical Takeaways

- Security should be built into systems, not bolted on afterward.
- Users should not have to compensate for insecure defaults.
- For personal AI workflows, secure-by-design means reducing sensitive data exposure, using least privilege, and checking outputs before acting.

## Personal Translation

When building or automating with AI:

- Do not paste secrets into models.
- Do not give tools broader file or account access than needed.
- Review generated code for risky behavior.
- Keep credentials in environment variables or secret stores.
- Prefer reversible actions while prototyping.
