## Workflow: Friction to System

Trigger: I notice the same annoying task twice.

Prompt:

```text
I keep getting friction around [task]. Help me turn it into a repeatable system. Identify the trigger, input, output, steps, checklist, and what could be automated later.
```

Output:

- Checklist
- Saved prompt
- Possible automation

## Workflow: Source-Backed Answer

Trigger: I need an answer that could be wrong if outdated.

Prompt:

```text
Help me answer this using sources. First list the claims we need to verify, then identify authoritative sources, then produce an answer with uncertainty clearly labeled.
```

Review:

- Sources included
- Dates checked
- Claims separated from interpretation

## Workflow: Learn by Building

Trigger: I want to learn a skill.

Prompt:

```text
Design a tiny project that teaches me [skill]. It should take 1-3 hours, include clear acceptance criteria, and produce a visible artifact.
```

## Workflow: Decision Partner

Trigger: I am stuck between options.

Prompt:

```text
Help me make this decision: [decision]. Options: [options]. Criteria: [criteria]. Give me tradeoffs, hidden risks, reversible vs irreversible parts, and a recommendation with confidence level.
```
