# AI Coding Workflow

Use AI as a pair programmer, reviewer, debugger, and teacher.

Loop:

1. Explain the goal and repo context.
2. Ask AI to inspect before editing.
3. Make small changes.
4. Run tests or build.
5. Ask for review of the diff.
6. Save patterns into [[AI Workflow Library]].

Prompts:

```text
Read this code and explain the architecture before suggesting changes.
```

```text
Find the smallest change that fixes this bug. Include likely tests.
```

```text
Review this diff for regressions, edge cases, and missing tests.
```
