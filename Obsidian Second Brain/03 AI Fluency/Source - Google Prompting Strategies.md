# Source - Google Prompting Strategies

Source: Google AI for Developers, Gemini API prompting strategies.  
URL: https://ai.google.dev/gemini-api/docs/prompting-strategies  
Accessed: 2026-06-24

## Practical Takeaways

- Be specific about what you want.
- Include examples when format matters.
- Break down complex tasks.
- Experiment and iterate.
- Ask for structured formats when outputs need to be reused.

## How I Use This

- Convert vague goals into explicit tasks.
- Use examples from my own vault when I want the model to match my style.
- For repeatable outputs, request tables, JSON, checklists, or templates.
