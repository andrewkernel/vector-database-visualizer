# Day-to-Day AI Use Cases

This note turns AI fluency into everyday action.

## Morning

- Turn vague intentions into a realistic plan.
- Ask what should be deleted, delegated, delayed, or done now.
- Convert goals into a first 10-minute action.

Prompt:

```text
Here is my day, energy, and current goals. Help me pick the highest-leverage actions and design a realistic schedule. Keep it honest, not motivational.
```

## Learning

- Explain a concept at three levels.
- Quiz you Socratically.
- Generate examples and counterexamples.
- Turn notes into flashcards.

## Coding

- Explain unfamiliar code.
- Design a small implementation plan.
- Generate tests.
- Review diffs.
- Debug error messages.

## Communication

- Draft concise messages.
- Make writing clearer.
- Prepare for hard conversations.
- Turn rambling thoughts into a respectful ask.

## Creativity

- Generate variations.
- Combine interests.
- Make outlines.
- Critique drafts.

## Administration

- Summarize documents.
- Extract tasks.
- Create checklists.
- Draft status updates.
