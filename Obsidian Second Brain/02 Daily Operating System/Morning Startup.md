# Morning Startup

Use this when you sit down to work.

1. Open [[Daily Dashboard]]
2. Check [[Goal Stack]]
3. Choose one meaningful priority
4. Ask:
   - What is the smallest useful next action?
   - What would make this easier?
   - Where can AI save time or increase quality?
5. Start a 25-minute focus block

AI startup prompt:

```text
Act as my daily planning partner. My current goals are [paste goals]. My available time and energy today are [describe]. Help me choose the highest-leverage 3 outcomes, identify one task to delegate to AI, and define the first 10-minute action.
```
