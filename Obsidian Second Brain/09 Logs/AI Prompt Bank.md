# AI Prompt Bank

## Plan

```text
Help me break this goal into milestones, weekly actions, and the next 10-minute step.
```

## Learn

```text
Teach me this concept with examples, then quiz me until I can explain it simply.
```

## Build

```text
Given this goal and code context, propose the smallest useful implementation plan and likely tests.
```

## Review

```text
Review this output for errors, unsupported claims, missing edge cases, and ways to make it clearer.
```

## Reflect

```text
Based on these notes, identify patterns, lessons, and one behavior change for tomorrow.
```

## Context Builder

```text
Before answering, tell me what context would most improve your answer. Then proceed using only what I provided and label assumptions.
```

## Taste Extractor

```text
Based on these examples I like and dislike, infer my taste criteria. Then use those criteria to improve the next version.
```

## Debug My Thinking

```text
Find the weak assumptions, missing evidence, and possible blind spots in my current plan. Be direct but practical.
```

## Turn Notes Into Action

```text
Convert these notes into: 1) decisions, 2) tasks, 3) questions, 4) reusable knowledge, and 5) things to delete.
```

## Explain Like a Builder

```text
Explain this concept by showing how I would use it in a real project. Include a tiny example, common failure modes, and one practice exercise.
```
