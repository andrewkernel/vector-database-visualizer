## Practical Takeaways

- Graph view visualizes relationships between notes in a vault.
- Notes are nodes and links are edges.
- Local graph helps inspect notes connected to the active note.
- Graph settings can filter, group, and adjust display.

## How I Use This

- Use global graph weekly to see the whole vault.
- Use local graph when working inside a project or topic.
- If a note is isolated, either link it or decide it is not useful.
