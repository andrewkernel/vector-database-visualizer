# Learning Science

Useful principles:

- Retrieval beats rereading
- Spacing beats cramming
- Projects create context
- Teaching exposes gaps
- Feedback accelerates skill

AI learning prompt:

```text
Quiz me Socratically on this note. Start easy, then increase difficulty. After each answer, give concise feedback.
```

Related: [[Learning Map]], [[Skill Tree]]

## Learning Protocol

1. Choose a skill from [[Skill Tree]].
2. Ask AI for a tiny project.
3. Study just enough to start.
4. Build something.
5. Retrieve from memory.
6. Get feedback.
7. Save the lesson in your own words.

## Techniques

Retrieval:

- Close the note and explain from memory.

Spacing:

- Revisit tomorrow, next week, next month.

Interleaving:

- Mix related skills instead of doing one forever.

Feedback:

- Ask AI, tests, peers, or reality.

Transfer:

- Use the idea in a different context.
