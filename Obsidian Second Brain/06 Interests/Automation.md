# Automation

Automation is best after the manual workflow is understood.

Good automation targets:

- Repeated
- Boring
- Rule-like
- Easy to verify
- Low downside if wrong

Related: [[AI Automation Ideas]], [[Personal AI Stack]]

## Automation Ladder

Level 1: Checklist

- Write the steps clearly.
- Use AI to improve the sequence.

Level 2: Template

- Create a reusable prompt, document, or note.

Level 3: Script

- Use PowerShell, Python, or JavaScript to automate local work.

Level 4: Connected Workflow

- Connect apps with APIs, webhooks, or automation platforms.

Level 5: Agentic Workflow

- AI chooses steps and uses tools under constraints.

## Automation Safety

- Start read-only.
- Log actions.
- Keep human approval for sending, deleting, buying, or posting.
- Test on sample data.
- Make failure obvious.
