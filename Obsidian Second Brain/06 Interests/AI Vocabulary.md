# AI Vocabulary

Use this as a plain-English glossary.

- LLM: large language model trained to generate and transform text.
- Token: chunk of text the model reads or writes.
- Context window: amount of information the model can consider at once.
- Prompt: instructions and context given to the model.
- System prompt: higher-priority instructions that shape behavior.
- Embedding: numeric representation of meaning used for search and retrieval.
- RAG: retrieval-augmented generation; looking up relevant information before answering.
- Agent: AI system that can plan, use tools, and take steps toward a goal.
- Tool use: model calls external functions such as search, files, code, or browser actions.
- Structured output: response constrained to a schema such as JSON.
- Evaluation: process for measuring whether AI outputs meet a standard.
- Hallucination: unsupported or false output presented as if true.
- Latency: how long the model takes to respond.
- Inference: running a model to produce output.
