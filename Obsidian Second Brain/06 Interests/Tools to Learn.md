# Tools to Learn

This is a living queue of tools that support AI fluency.

## Core

- Obsidian: notes, links, graph, canvas
- Git and GitHub: version control and proof of work
- Python: automation, scripts, data work
- JavaScript: web apps and interfaces
- VS Code or similar editor: coding workflow

## AI

- ChatGPT or equivalent assistant
- Coding agents
- Search-enabled AI
- Image generation tools
- Speech-to-text tools
- API playgrounds

## Automation

- PowerShell
- Node.js
- Zapier/Make/n8n style tools
- Browser automation
- Local scripts

## Learning Rule

Do not learn tools in isolation. Attach each tool to a project in [[Project Radar]].
