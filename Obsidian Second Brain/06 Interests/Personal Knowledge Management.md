# Personal Knowledge Management

PKM is the practice of capturing, connecting, and reusing knowledge.

This vault uses:

- Maps of content
- Daily notes
- Project notes
- Templates
- Reviews
- Dense links

Related: [[Personal Knowledge Graph Project]], [[Personal Operating System]]

## Note Types

Fleeting note:

- Raw capture in [[Inbox]]

Project note:

- Outcome, next action, log, links

Source note:

- Facts, takeaways, actions, source URL

Map note:

- Navigation hub for a domain

Workflow note:

- Trigger, input, prompt, output, review checklist

## Link Quality

A link should answer one of these:

- Supports
- Explains
- Depends on
- Contrasts with
- Applies to
- Leads to action
