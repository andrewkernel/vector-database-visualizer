Small AI-assisted projects to grow skill and proof of work.

## Weekend Projects

- Personal prompt library web app
- Daily note summarizer
- AI-assisted habit tracker
- Source-backed research note generator
- Local file organizer
- Coding error explainer

## Selection Filter

Pick ideas that:

- Solve a real annoyance
- Can be demoed visually
- Teach one new skill
- Produce reusable code or prompts
- Fit into one weekend
